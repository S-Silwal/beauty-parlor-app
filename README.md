# Beauty Parlor Web Application

A modern full-stack web application for managing a beauty parlor business. Customers can book appointments, while staff and admins can manage bookings, services, and users.

---

## ✨ Features

- **User Authentication** (Register, Login, MFA, Password Reset)
- Role-based Access Control (Customer, Staff, Admin)
- Appointment Booking System
- Service Management
- Admin Dashboard
- Responsive UI

---

## 🛠️ Tech Stack

| Layer       | Technology                  |
|-------------|-----------------------------|
| Frontend    | Next.js 14 + TypeScript + Tailwind CSS |
| Backend     | Node.js + Express + TypeScript |
| Database    | PostgreSQL + Prisma         |
| Authentication | JWT + Refresh Tokens + bcrypt |
| Email       | SendGrid (for OTP & Reset)  |
